    修饰器作为 Python 著名的语法糖，可以说是 Python 语言的一大精髓了，尽管日常任务中其实用的比较少，但是你如果需要开发一个 Python 库的话，那就很有必要学习。

    1. 装饰器简介

    装饰器(修饰器) 是一个非常著名的设计模式，经常被用于有切面需求的场景，较为经典的有插入日志、性能测试、事务处理等。装饰器是解决这类问题的绝佳设计，有了装饰器，我们就可以抽离出大量函数中与函数功能本身无关的雷同代码并继续重用。概括的讲，装饰器的作用就是为已经存在的对象添加额外的功能。

    另外，装饰器有一个非常洋气的名字，叫做“闭包”，这在很多语言中都会涉及闭包的概念。

    2. 函数装饰器

    在 StackFlow 上有这样一个问题，如何通过以下的代码实现输出 <b><i>Hello</i></b>。

    @makebold  
@makeitalic  
def say():  
   return "Hello"

    实现方式很简单，具体如下：

    def makebold(func):
    def wrapper():
        res = '<b>' + func() + '</b>'
        return res
    return wrapper


def makeitalic(func):
    def wrapper():
        res = '<i>' + func() + '</i>'
        return res
    return wrapper


@makebold
@makeitalic
def say():
    return "hello"

print(say())
# <b><i>Hello</i></b>

    2.1 内置修饰器

    Python 也有 3 个内置修饰器，分别为 staticmethod、classmethod 和 property，作用分别是把类中定义的实例方法变成静态方方法、类方法和类属性。关于这三种内置修饰器的使用，我会在后续专门写一篇文章进行介绍。

    2.2 对一个参数的函数进行修饰

    如果原函数带有参数，那么闭包函数必须保持参数一致，并且将参数传递给原方法。

    def w_hello(func):
    def wrapper(name):
        func(name)
    return wrapper


@w_hello
def hello(name):
    print("Hello, %s" % name)


hello("yumingmin")
# Hello, yumingmin

    2.3 对多个参数的函数进行修饰

    对于有多个参数的函数进行闭包，则可以使用 *args 和 *kwargs* ，其实也是非常简单。

    def w_hello(func):
    def wrapper(*agrs, **kwargs):
        func(*agrs, **kwargs)
    return wrapper


@w_hello
def hello(firstname, lastname):
    print("Hello, %s-%s. Welcome!" % (firstname, lastname))


hello("yu", "mingmin")
# Hello, yu-mingmin. Welcome!

    2.4 带返回值的函数进行装饰

    对于带有返回值的函数，其实只需要在闭包函数中用一个变量进行接收，返回该变量即可。

    def w_hello(func):
    def wrapper(*agrs, **kwargs):
        res = func(*agrs, **kwargs)
        return res
    return wrapper


@w_hello
def hello(firstname, lastname):
    return "Hello, %s-%s. Welcome!" % (firstname, lastname)


print(hello("yu", "mingmin"))
# Hello, yu-mingmin. Welcome!

    2.5 给装饰器添加参数

    说了这么多，上述例子闭包装饰器只能接收函数名，那么可以它自己本身可以接收参数么，答案是可以的。废话不多说，直接上代码。

    def visitor(name='guest'):
    def visit_log(func):
        def wrapper(*args, **kwargs):
            print("logging: visitor is %s" % name)
            res = func(*args, **kwargs)
            return res
        return wrapper
    return visit_log


@visitor(name='yumingmin')
def hello(firstname, lastname):
    return "Hello, %s-%s. Welcome!" % (firstname, lastname)


print(hello("yu", "mingmin"))

    输出结果：

    logging: visitor is yumingmin
Hello, yu-mingmin. Welcome!

    既然我们在闭包装饰器函数中指定了默认的参数 name 为 guest，那么可以在使用 @visitor 时不添加 name="yumingmin"，让它默认输出 logging: visitor is guest 么？答案是不可以的，这样使用会报错。

    @visitor
def hello(firstname, lastname):
    return "Hello, %s-%s. Welcome!" % (firstname, lastname)

print(hello("yu", "mingmin"))

    报错信息为：

    TypeError: visit_log() takes 1 positional argument but 2 were given

    显然，hello 函数其实被当做参数传给了 name 参数，所以报错。

    你想要实现输上述默认参数 name 的输出，可以采用以下方式：

    @visitor()
def hello(firstname, lastname):
    return "Hello, %s-%s. Welcome!" % (firstname, lastname)

print(hello("yu", "mingmin"))

    输出结果：

    logging: visitor is guest
Hello, yu-mingmin. Welcome!

    2.6 装饰器带类参数

    装饰器带类参数，指定的参数必须为 cls。以下为具体示例：

    class Locker:
    def __init__(self):
        print("locker.__init__() should be not called.")

    @staticmethod
    def acquire():
        print("locker.acquire() called.（这是静态方法）")

    @staticmethod
    def release():
        print("  locker.release() called.（不需要对象实例）")


def deco(cls):
    """cls 必须实现acquire和release静态方法"""
    def _deco(func):
        def __deco():
            print("before %s called [%s]." % (func.__name__, cls))
            cls.acquire()
            try:
                return func()
            finally:
                cls.release()
        return __deco
    return _deco


@deco(Locker)
def myfunc():
    print(" myfunc() called.")


myfunc()

    输出结果：

    before myfunc called [<class '__main__.Locker'>].
locker.acquire() called.（这是静态方法）
 myfunc() called.
  locker.release() called.（不需要对象实例）

    3. 类装饰器

    当创建一个类对象后，直接去执行这个对象，因为它不是 callable，无法直接执行。但进行修改后，就可以直接执行调用。

    class Test:
    def __call__(self, *args, **kwargs):
        print("call called")

t = Test()
print(t())
# call called
# None

    使用闭包类对函数进行装饰：

    class Test:
    def __init__(self, func):
        print('test init')
        print('func name is %s ' % func.__name__)
        self.__func = func

    def __call__(self, *args, **kwargs):
        print('this is wrapper')
        self.__func()


@Test
def test():
    print('this is test func')


test()

    输出结果：

    test init
func name is test 
this is wrapper
this is test func

    4. 函数装饰器实例

    4.1 性能测试——运行时间装饰器

    import time


def timeit(func):
    """装饰器，输出 func 函数运行的时间"""
    def wrapper():
        start = time.clock()
        res = func()
        end = time.clock()
        print("funcname {0} cost time: {1}".format(func.__name__, (end - start)))

        return res

    return wrapper


@timeit
def run1():
    a = [i for i in range(100000000)]


@timeit
def run2():
    return [i for i in range(10000000)]    


print(run1())
print(run2()[:5])

    输出结果:

    funcname run1 cost time: 0.8060808675805581
None
funcname run2 cost time: 0.6412041402756763
[0, 1, 2, 3, 4]

    这里还可以引入 wraps 函数，来确保被装饰的函数 __name__ 的属性不变。

    import time
from functools import wraps


def timeit(func):
    """装饰器，输出 func 函数运行的时间"""
    @wraps(func)
    def wrapper():
        start = time.clock()
        res = func()
        end = time.clock()
        print("funcname {0} cost time: {1}".format(func.__name__, (end - start)))

        return res

    return wrapper

    4.2 将字典数据转换为JSON格式

    很多时候我们通过 API 接口 POST 数据时，我们需要将数据转换成 JSON 格式，而我们的数据格式是 dict 格式的。这就可以通过创建一个修饰器，来让你的函数输出为 JSON 格式。以下为代码示例：

    import json
from functools import wraps


def json_converter(func):
    """将被装饰函数的返回结果转换成JSON格式"""
    @wraps(func)
    def wrapper(*args, **kwargs):
        """闭包函数"""
        result = func(*args, **kwargs)
        json_result = json.dumps(result)
        return json_result
    return wrapper


@json_converter
def my_test():
    # 创建字典类型返回值
    return dict(name="Ammy", passwd=666666, age=16)


json_res = my_test()
print("被装饰后返回结果: \n", json_res)

    输出结果：

    被装饰后返回结果: 
 {"name": "Ammy", "passwd": 666666, "age": 16}
