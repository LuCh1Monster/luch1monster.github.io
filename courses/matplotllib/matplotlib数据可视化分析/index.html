    8. 箱形图

    8.1 概念

    • 箱形图（Box-plot）,又称盒须图、盒式图或者箱线图
    • 是一种用作显示一组数据分散情况资料的统计图
    • 上边缘，上四分位，中位数，下四分位，下边缘，异常值

    8.2 简单的例子

    import matplotlib.pyplot as plt
import numpy as np

np.random.seed(100)

plt.boxplot(data)
plt.show()

    • np.random.seed

    seed() 用于指定随机数生成时所用算法开始的整数值，如果使用相同的 seed 值,则每次生成的随机数都相同。如果不设置这个值，则系统根据时间自己来选择这个值，此时每次生成的随机数因时间差异而不同。

    • np.random.normal
      • loc : 概率分布的均值，对应整个分布的中心 center
      • scale : 概率分布的标准差，对应于分布的宽度，scale 越大越矮胖；scale 越小，越瘦高
      • size : 输出的 shape，默认是 None，只输出一个值

    10. 面向对象 VS. Matlab Style

    10.1 三种方式简介

    • pyplot : 经典高层封装，到目前为止，我们所用的都是 pyplot
    • pylab : 将 Matplotlib 和 Numpy 合并的模块，模拟 matplotlib 的编程环境
    • 面向对象的方式，Matplotlib 的精髓，更基础和底层的方式

    10.2 三种方式的优劣

    • pyplot : 简单易用，交互使用时方便，可以根据命令实时作图，但底层定制能力不足
    • pylab : 完全封装，环境最接近 matlab, 不推荐使用
    • 面向对象(object-oriented): 接近 matplotlib 基础和底层的方式，难度稍大，但定制能力强，是matplotlib 中精髓

    总结: 实战中推荐，根据需求，综合使用 pyplot 和面向对象的方式，显式地导入 numpy
