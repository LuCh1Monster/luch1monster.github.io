    关于 Python 连接 impala 的教程已经在之前讲过，本篇主要讲一下如何通过 Python 如何在 impala 中建表的操作，用过 impyla 的同学应该知道，这个包是不支持将 Dataframe 在 impala 中建表的,，它只支持执行 SQL 语句。但是，我们从 impyla 中拉取数据后，经过一些列数据处理，模型打分，最终得到一个 Dataframe，但是最终我们要把这个表存入 impala，供被人使用。

    这个问题通常有2种解决办法，下面我们分别介绍:

    1. INSERT语句建表

    将你 Dataframe 转化为 sql 插入语句，然后利用 impyla 的 curor.execute(sql) 来完成建表的工作，具体示例如下:

    假设你的 Dataframe 是像下面这样:

    a b c
    1 1 2 3
    2 4 5 6

    那么首先需要在 impala 执行一段表定义(DDL)语句，即:

    DROP TABLE IF EXISTS test.t1
;
CREATE TABLE IF NOT EXISTS test.t1 
(a INT,
 b INT,
 c INT)
 ;

    然后利用 python 将你的 Dataframe 转换为 SQL 插入语句，并用 cursor.execute 执行该 sql 语句:

    for raw in df.values:
    value = '{!r}'.format(list(raw))
    value = '(' + value.strip('[').strip(']') + ')'
    values_list.append(value)

values = ','.join(x for x in values_list)
insert_sql = """insert into test.t1 values %s""" % values
cursor.execute(insert_sql)

    这样就完成了将 Dataframe 存入 impala 中，但是 insert 的方式建表会产生很多小文件，且非常的不方便。下面我们讲一下第2种方式

    2. 通过HDFS、Hive 和 Impala 建表

    这种方式的核心思想是将 Dataframe 先在本地存成 CSV 文件，然后通过 Python 连接 HDFS 将 CSV 文件 PUT 上去，再通过Hive 执行创建外部表，最后将 Hive 数据同步到 impala 中，这种建表方式不会产生小文件，且后期维护较为简单。

    2.1 Python连接HDFS

    Python 连接 HDFS 主要有两个包: pyhdfs 和 hdfs。注意: pyhdfs 的版本一定不要是 0.2.2，该版本经常时好时坏，不建议使用，可以选用 0.2.1 版本。

    → pip install pyhdfs==0.2.1
→ pip install hdfs

    第一种是使用 hdfs 的示例代码:

    from hdfs import InsecureClient
import os

def create_hdfs_dir(hdfs_path, host='http://ip:port', user='hive'):

    hdfs_conn = InsecureClient(url=host, user=user)
    response = hdfs_conn.makedirs(hdfs_path)

    return response

def upload_to_hdfs(file_path, hdfs_path, 
                   host='http://ip:port', 
                   user='hive', overwrite=True):

    hdfs_conn = InsecureClient(url=host, user=user)
    response = hdfs_conn.upload(hdfs_path, file_path, overwrite=overwrite)

    return response

def download_from_hdfs(hdfs_path, 
                       save_path=None, 
                       host='http://ip:port', 
                       user='hive', overwrite=True):

    if save_path is None:
        save_path = os.path.join(os.getcwd(), 'tmp_folder')     # 存放文件的临时目录

    # 建立存放文件的文件夹，若文件夹已存在则清空其中的文件
    if not os.path.exists(save_path):
        os.mkdir(save_path)
    else:
        if overwrite:
            for f in os.listdir(save_path):
                os.remove(os.path.join(save_path, f))

    # 从HDFS目录下载文件
    hdfs_conn = InsecureClient(url=host, user=user)
    start_time = datetime.now()
    cnt = 0
    for f in hdfs_conn.list(hdfs_path):
        if hdfs_conn.status(os.path.join(hdfs_path, f))['type'] == 'FILE':
            hdfs_conn.download(os.path.join(hdfs_path, f), os.path.join(save_path, f))
            cnt += 1
    print('Download {0} files, Time cost: {1}'.format(cnt, datetime.now() - start_time))
