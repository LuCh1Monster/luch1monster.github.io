    1. wxPytho介绍与安装

    1.1 wxPython 介绍

    • wxPython 是 wxWidgets 在 Python 语言下的封装
    • wxWidgets 是一个跨平台的 GUI 应用编程接口，使用C++编写
    • 塑像即所得，缺乏成熟的 IDE
    • 图形界面风格与系统的风格相似

    1.2 wxPython VS. PyQt

    为什么选择 wxPython 而不是 PyQt ?

    • 协议
    • 编码风格
    • 复杂性
    • 个人习惯

    1.3 wxPython 安装

    • 官网地址: http://www.wxpython.org/
    • 下载地址: http://www.wxpython.org/download.php#msw

    安装程序自动寻找 Python 的位置，基本一路下一步。

    注：python(x,y) 已经安装了 wxPython

    参考资料: http://wiki.wxpython.org/

    图形界面: http://wiki.wxpython.org/SizerTutorials

    2.2 绝对布局

    • 基于控件的坐标摆放空间
    • 简单直观
    • 方法单一
    • 不能随窗口的改变而调整位置
    import wx

class Example(wx.Frame):

    def __init__(self):
        wx.Frame.__init__(self, parent=None, id=-1, title="hello wx", size=(600, 600))
        panel = wx.Panel(self, -1)
        self.Center()

        # 按钮
        button = wx.Button(panel, label=u"上海理工大学", size=(100, 100), pos=(20, 0))

        # 静态文本
        staticText = wx.StaticText(panel, -1, u"郁明敏", size=(100,20), pos=(20, 120))
        # 单行可编辑文本框
        text = wx.TextCtrl(panel, -1, u"可编辑文本", size=(100, 20), pos=(20, 160))
        # 密码文本框
        password = wx.TextCtrl(panel, -1, u"输入密码", style=wx.TE_PASSWORD, pos=(20, 200))
        # 多行文本框
        mutiText = wx.TextCtrl(panel, -1, u"输入多行\n文本", style=wx.TE_MULTILINE, pos=(20, 240))

        # 复选框
        checkBox1 = wx.CheckBox(panel, -1, u"苹果", pos=(150, 20))
        checkBox2 = wx.CheckBox(panel, -1, u"梨子", pos=(150, 40))

        # 单选框
        radio1 = wx.RadioButton(panel, -1, u"男", pos=(150, 80), style=wx.RB_GROUP)
        radio2 = wx.RadioButton(panel, -1, u"女", pos=(150, 100))

        # 单选按钮框
        radioList = [u"python", u"spark", u"hadoop", u"hive"]
        wx.RadioBox(panel, -1, u"大数据", (220, 0), wx.DefaultSize, radioList, 2, wx.RA_SPECIFY_ROWS)
        wx.RadioBox(panel, -1, u"大数据", (220, 100), wx.DefaultSize, radioList, 1, wx.RA_SPECIFY_ROWS)
        wx.RadioBox(panel, -1, u"大数据", (220, 180), wx.DefaultSize, radioList, 1, wx.RA_SPECIFY_COLS)

        # 列表框
        name = [u"路飞", u"索隆", u"娜美", u"罗宾"]

    4. 多线程与事件

    4.1 Python多线程

    • 不使用多线程可能会导致图形界面卡死

    • 多线程可以让程序以更高的效率运行

    • 多线程可以让程序作更多的事情

    • thread.start_new_thread(函数名, (参数1, 参数2, …))

    • 使用 Threading 模块创建线程(不讲)

    4.2 控件的事件

    • 按键点击

    • 文本域内容改变

    • 鼠标滑过

    • 键盘按下

    4.4 代码解析

    4.4.1 绑定触发事件

    self.Bind(wx.EVT_BUTTON, self.onSend, self.send)
self.Bind(wx.EVT_BUTTON, self.onClear, self.clear)
self.Bind(wx.EVT_BUTTON, self.onScreen, self.screen)

    将控件 send、clear、screen 绑定到按钮事件 onSend、onClear、onScreen。注: onSend、onClear、onScreen 均为构建的类方法。

    4.4.2 触发事件

    1) _onSend事件

    _onSend 触发事件是由 onSend 事件触发。

    def _onSend(self, event):
    if self.lesson.GetSelection() != -1:
        lesson = self.lessonList[self.lesson.GetSelection()]
    else:
        lesson = u"未选择课程"

    command = self.commandText.GetValue()
    writePy = self.writePyText.GetValue()

    print(u"选中的课程: %s" % lesson)
    print(u"执行的命令是: %s" % command)
    print(u"写入Python的代码: %s" % writePy)

    GetSelection() : 返回的是选中列表框的第几项，即索引值。如果返回的是 -1，表示类表框没有任何内容被选中。

    GetValue() : 返回的获取文本框的内容。

    2) onClear 事件

    onClear 事件是由点击清空命令按钮触发。

    def onClear(self, event):
    self.commandText.Clear()
    self.writePyText.Clear()
    self.writePyText.AppendText(u"# -*- coding:utf8 -*-\n#早这里输入Python代码")

    commandText.Clear() : 清除文本框内的内容

    AppendText : 添加内容到文本框中

    3) onScreen 事件

    onScreen 事件由到点击查看屏幕按钮触发。

    def onScreen(self, event):
    img = wx.Image(r"xiaoxin.jpg", wx.BITMAP_TYPE_ANY).Scale(300, 200)
    self.screenBox.SetBitmap(wx.BitmapFromImage(img))
    self.gridBagSizerAll.Fit(self)

    4) onSend 事件

    onSend 事件由点击发送命令触发。

    def onSend(self, event):
    import thread
    thread.start_new_thread(self._onSend, (event, ))

    • 使用多线程

    • (event, ) 如果没有参数，就一定要有逗号，

    • 调用 _onSend 函数

    • 这个多线程使用有很大问题，运行容易卡死

    可以在 _onSend 方法添加 time.sleep(0.5) 来对操作进行延时，防止操作之间的引起图形界面的卡死。

    def _onSend(self, event):

    time.sleep(0.5)
    if self.lesson.GetSelection() != -1:
        lesson = self.lessonList[self.lesson.GetSelection()]
    else:
        lesson = u"未选择课程"

    command = self.commandText.GetValue()
    writePy = self.writePyText.GetValue()

    print(u"选中的课程: %s" % lesson)
    print(u"执行的命令是: %s" % command)
    print(u"写入Python的代码: %s" % writePy)
