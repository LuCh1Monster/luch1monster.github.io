    1. Scala简介

    1.1 概述

    对于初学者来说，学一种语言或工具，首先关心的学习该语言或工具的意义、特点及作用等。

    Scala 是一种有趣的语言。它一方面吸收继承了多种语言中的优秀特性，一方面又没有抛弃 Java 这个强大的平台，它运行在 Java 虚拟机 (Java Virtual Machine) 之上，轻松实现和丰富的 Java 类库互联互通。它既支持面向对象的编程方式，又支持函数式编程。它写出的程序像动态语言一样简洁，但事实上它确是严格意义上的静态语言。Scala将过去几十年计算机语言发展历史中的精萃集于一身，化繁为简，为程序员们提供了一种新的选择。

    目前大数据运算架构 Spark 就是用 Scala 编写的，在 Spark 中把 scala 的分布式、高并发、简洁等特性发挥得淋漓尽致。

    1.2 简介

    Scala 是 Scalable Language 的简写，是联邦理工学院洛桑(EPFL) 的 Martin Odersky 于2001年基于 Funnel 的工作开始设计的。Funnel是把函数式编程思想和 Petri 网相结合的一种编程语言。Odersky 先前的工作是 Generic Java 和 javac(Sun Java编译器)。

    Java 平台的 Scala 于 2003 年底 / 2004年初发布。.NET 平台的 Scala 发布于 2004 年 6 月。该语言第二个版本，v2.0，发布于 2006 年3月。截至目前，最新版本是版本 2.11。

    Scala主要特性:

    • 面向对象

    Scala 是一种纯面向对象的语言，每个值都是对象。

    • 函数式编程

    Scala 也是一种函数式语言，其函数也能当成值来使用。Scala 提供了轻量级的语法用以定义匿名函数，惰性求值，支持高阶函数，允许嵌套多层函数，并支持柯里化。Scala 的 case class 及其内置的模式匹配相当于函数式编程语言中常用的代数类型。

    • 静态类型

    Scala 具备类型系统，通过编译时检查，保证代码的安全性和一致性。

    • 扩展性

    Scala 提供了许多独特的语言机制，可以以库的形式轻易无缝添加新的语言结构。

    • 并发性

    Scala 使用 Actor 作为其并发模型，Actor 是类似线程的实体，通过邮箱发收消息。Actor 可以复用线程，因此可以在程序中可以使用数百万个 Actor，而线程只能创建数千个。在 2.10 之后的版本中，使用 Akka 作为其默认 Actor 实现。

    1.3 Scala与Java、Python

    Scala 与 Java 异同:

    • 两者都是基于基于 JVM 及 JAVA 的生态系统。

    • scala 不需要分号结尾，return 可以省略。

    Scala 与 Python 异同:

    • 两者都是面向对象，也拥有一个 REPL。

    • 函数定义都以 def 开头。

    • Scala 不像 Python 一样通过代码缩进来表示代码的层次关系，而是和通常的语言一样使用 {} 来表示代码的层级。

    • scala 缺少 python 那样丰富的数据处理，python 不是为大数据设计的，scala 可以说是大数据导向的。

    1.4 安装配置

    Scala 是基于 java 之上，大量使用 java 的类库和变量，必须使用 Scala 之前必须先安装 Java。

    第1步:安装配置Java

    下载安装 JDK 1.6 以上版本，设置 JAVA_HOME 环境变量及 JDK 的 bin 目录。

    export JAVA_HOME=/usr/local/java
export PATH=$PATH:$JAVA_HOME/bin

    第2步: 安装配置Scala

    可以从 Scala 官网地址 http://www.scala-lang.org/downloads 下载 Scala 二进制包。

    • 解包到指定目录下
    → tar zxvf scala-2.11.8.tgz -C /usr/local/
    • 重命名
    → mv scala-2.11.8 scala
    • 在 /etc/profile 配置文件中，追加如下内容
    export SCALA_HOME=/scala
export PATH=$PATH:$SCALA_HOME/bin

    然后 source /etc/profile 是该配置文件立即生效，运行 scala 命令，查看 scala 安装是否成功。

    1.5 小试牛刀

    终端出现了 scala> 提示符，和 Python 的 REPL 一样，Scala 也有一个 REPL。现在我们可以开始写一些语句或编写一些代码，进行体验或测试。

    scala> 4 + 5
res3: Int = 9  // res3是解释器自动生成的变量名，后面可直接使用该变量

scala> res3 * 10
res4: Int = 90

scala> res4 + 100
res5: Int = 190

    scala 是一种伸缩性很强的语句，尤其是适合于函数式编程，可以像 python 一样作为脚本语言来执行，如下例。

    [hadoop@master ~]→ cat test.scala
#!/bin/env scala
println("hello " + args(0) + "!")

[hadoop@master ~]→ ./test.scala 高峰
hello 高峰!

    2. 基础知识

    2.1 常量和变量

    Scala 中常量用 val 定义，变量以 var 定义

    • val(value) 表示常量，为不可变的，不可修改，但在 REPL 可以重新赋值。

    • var(variable) 表示可变的，可以重新赋值或修改

    定义变量时，可不显式指定数据类型，scala 有很强类型自动推导功能，Scala 具有动态语言似的编写代码，但又有静态语言的编译时检查。

    scala> val a=10
a: Int = 10

scala> val b=10.5
b: Double = 10.5

scala> b=12.5   //b是常量类型，不可修改，否则报错。
<console>:8: error: reassignment to val
       b=12.5
        ^

scala> var c=80
c: Int = 80

scala> var d:Int=100
d: Int = 100

scala> d=200
d: Int = 200

    2.2 基本类型

    Scala 中基础数据类型有: Byte、 Short、 Int、 Long、 Float、 Double、 Boolean、 Char 和 String。和 Java 不同的是，Scala 中没在区分原生类型和装箱类型，如 int 和 Integer。它统一抽象成 int 类型，这样 Scala 中所有类型都是对象了。

    • 字符串

    Scala 中单引号和双引号是有区别的，单引号用于字符，双引号用于字符串。而且双引号中的特殊字符，需要转义字符 "\"。此外，还有多行字面量(如 3 个引号)和字符串内插等特性。

    Scala 是基于 JVM 平台，默认使用 unicode，所以变量名是可以直接使用中文的。而在 scala 中，中文也会直接显示，不会像 python2 一样输出成 unicode 编码形式: \u***。

    scala> val s = 'h'
s: Char = h

scala> var s1 = "hello world!"
s1: String = hello world!

scala> var s1 = "hello \nworld!"
s1: String =
hello
world!

scala> var d = """hello \nworld!"""     // 多行字面量，内部字符都视为原始字符
d: String = hello \nworld!

scala> val name = "scala"
name: String = scala

scala> println(s"the tool is $name")     // 字符串前加s，可以内插字符串
the tool is scala

    3 个双引号在 scala 中也有特殊很含义，它代表被包括的内容是原始字符串，可以不需要字符转码。这一特性在定义正则表达式很有优势。

    2.3 控制语句

    Scala 内置的控制语句有 if、while、for、match case 四大主要控制语句。Scala 的控制语句都是表达式，是有返回值的。

    • if 语句格式
    // 格式一
if (Boolean express) express
// 格式二
if (Boolean express) express else express

    示例:

    scala> val a = 10; val b = 15
a: Int = 10
b: Int = 15

scala> val c = if (a > b) a else b
c: Int = 15
    • while 语句格式
    while (Boolean express){
    statement
}

    示例:

    scala> var x = 2;
x: Int = 2

scala> while(x > 0) {
     | x -= 1
     | println(x)
     | }
1
0
    • for 语句格式
    for (identifies <- iterator) [yield] [express]

    关键字 yield 是可选的如果表达式指定这个关键字，所有表达式的值将作为一个集合返回；如果没有指定这个关键字，则调用表达式，但不能访问其返回值。

    示例:

    scala> val list = List(1,2,3)
list: List[Int] = List(1, 2, 3)

scala> for(i <- list) println(i)
1
2
3

scala> for(j <- 1 to 5) yield j * 2
res3: scala.collection.immutable.IndexedSeq[Int] = Vector(2, 4, 6, 8, 10)

scala> val list2 = for(j <- 1 to 5) yield j * 2
list2: scala.collection.immutable.IndexedSeq[Int] = Vector(2, 4, 6, 8, 10)

    Python 有个列表推导式，在 Scala 中也有类似功能，而且更强大，作用不限于列表，在整个集合(如 Seq、Map、Set、List、Array 等) 都适合。当然，在 Scala 这种方式叫做迭代器守卫(iterator guard)。

    迭代器守卫格式:

    for(identifies <- iterator if Boolean express) [yield] [express]

    示例:

    scala> val s1 = "python,ipython,pandas,scala"
s1: String = python,ipython,pandas,scala

scala> for(i <- s1.split(",") if i.size > 5) println(i)
python
ipython
pandas
    • match case 语句格式
    express match{
    case pattern_match => express
    [case statement]
}

    scala 中 match case 称之为模式匹配，它是默认 break 的，只要其中一个 case 语句匹配，就终止之后的所有比较。且对应的 case 语句的表达式值将作为整个 match case 表达式的值返回。

    示例:

    scala> val tools = "scala"
tools: String = scala

scala> val t1 = tools match{
     | case "python" => "python"
     | case "ipython" => "python"
     | case "pandas" => "python"
     | case _ => "others"
     | }
t1: String = others

    这种多条件对应一个结果的情况，可以使用竖线 | 来表示，这样结构更简洁。

    scala> val t1 = tools match{
     | case "python"|"ipython"|"pandas" => "python"
     | case "scala" => "scala"
     | case _ => "others"
     | }
t1: String = scala

    3. 常用集合

    在 Python 中，常用集合类型有: list、tuple、set、dict 等。Scala 中对应的有: Array、List、Tuple、Set、Map 等。Scala 中这些关键字的第一个字母需要大写。

    scala 中各集合间的层次结构如下:

    3.1 数组

    数组 Array，下标从 0 开始，通过 () 来引用，成员可修改。

    scala> val arr1 = Array("python", "java", "scala")
arr1: Array[String] = Array(python, java, scala)

scala> arr1(1) = "spark"
scala> arr1
res9: Array[String] = Array(python, spark, scala)

    3.2 数组

    列表 List，下标从 0 开始，通过 () 来引用，成员可读，但不可修改。List 是 Iterable 的一个子类型。

    scala> val list1 = List(10, 20, 5)
list1: List[Int] = List(10, 20, 5)

scala> list1(2)
res10: Int = 5

scala> list1(1) = 30
<console>:13: error: value update is not a member of List[Int]
       list1(1) = 30
       ^

    列表还可通过 :: 操作符，将一个元素和列表连接起来，并把元素放在列表的开头。

    scala> 1::list1
res12: List[Int] = List(1, 10, 20, 5)
    方法 示例 说明
    contains List(1,2,3).contains(2) 检查列表中是否包含某个元素
    exists List(1,2,3).exists(_ > 2) 检查列表中是否至少有一个满足条件的元素
    forall List(1,2,3).forall(_ > 2) 检查列表中所有元素都满足条件
    distinct List(1,2,3,1,2).distinct 去掉重复元素
    filter List(1,2,3,4).filter(_ > 2) 过滤使条件为 True 的元素
    flatten List(List(1,2), List(3,4)).flatten 将列表中的列表变为元素的列表
    map List(1,2,3,4).map(_ * 10) 将函数作用于遍历集合中每个元素得到新列表，有返回值
    foreach List(1,2,3,4).foreach(x => println(x+10)) 将函数作用与遍历集合中每个元素，无返回值
    reduce List(1,2,3,4).reduce(_ + _) 从列表第一个元素开始从左带头归约处理
    sortBy List("abc","cd","efg").sortBy(_.size) 根据函数返回值进行排序
    take List(1,2,34).take(2) 取列表前 n 个元素
    size List("abc", "cd", "efg").size 统计列表元素个数
    zip List(1,2) zip List(3,4) 将两个列表合并为一个元组列表，列表对应索引组成一个元组

    3.3 元组

    与 Python 中类似，Scala 也采用小括号来定义元组 Tuple，下标从 1 开始，通过 _ 下标来访问成员，不能通过 () 访问成员，这点与数组、列表不同，其成员可以是不同类型，元组最多只支持 22 个元素。

    scala> val t1 = (2, 5, 1, 4)
t1: (Int, Int, Int, Int) = (2, 5, 1, 4)
scala> t1._2
res16: Int = 5

    3.4 集合

    Set 是一个不重复且无序的集合，初始化一个集合需要使用 Set 对象。

    scala> val set1 = Set(1,2,2,4,5)
set1: scala.collection.immutable.Set[Int] = Set(1, 2, 4, 5)

    3.5 映射

    Scala 中的 Map 是个可变的键值库，其他语言中也称为离散映射(HashMap)、字典(dictionary)，与Python 中的字典 dict 类似。

    Scala并没有提供一个 {} 来定义 Map，创建 Map 时，指定键值对为元组，也可以使用关系操作符(->) 来指定键和值元组。与 Set、List 一样，Map 也是 Iterable 的一个子类型，支持与 List 相同操作。Map 缺省是不可修改的，如果要成为可变 Map，需要显式定义其类型，如 scala.collection.mutable.Map。

    scala> val m1 = Map(1 -> "python", 2 -> "java", 4 -> "scala")
m1: scala.collection.immutable.Map[Int,String] = Map(1 -> python, 2 -> java, 4 -> scala)

scala> m1(4)
res2: String = scala

scala> m1 += (3 -> "spark")     // 因 m1 缺省为不可变的，故添加元素报错
<console>:13: error: value += is not a member of scala.collection.immutable.Map[Int,String]
       m1 += (3->"spark")


scala> val m1 = scala.collection.mutable.Map(
        1 -> "python", 
        2 -> "java", 
        4 -> "scala")
m1: scala.collection.mutable.Map[Int,String] = Map(2 -> java, 4 -> scala, 1 -> python)

scala> m1 += (3 -> "spark")
res6: m1.type = Map(2 -> java, 4 -> scala, 1 -> python, 3 -> spark)

scala> m1(5) = "hadoop"
scala> m1
res8: scala.collection.mutable.Map[Int,String] = Map(2 -> java, 5 -> hadoop, 4 -> scala, 1 -> python, 3 -> spark)

    Map 用于循环语句

    scala> for(key <- m1.keys) print(s"$key")
25413

scala> for(value <- m1.values) print(s"$value ")
java hadoop scala python spark

    3.6、集合的模式匹配

    模式匹配是 scala 的一个核心特性，这个特性可用于 scala 的数据结构，利用该特性可大大简化代码、使逻辑显得更清晰。

    scala> val list1 = List("hadoop", "spark", "python")

scala> val rets = list1 match{
    case x if x contains("spark") => "bigdata tools"
    case _ => "no"
}
rets: String = bigdata tools

    4. 函数

    函数是可重用的逻辑单位，在 Scala 中，函数是一等公民，函数式编程是 Scala 的一大特色。函数可以被赋值给一个变量，也可以做为一个函数的参数被传入，甚至还可以做为函数的返回值返回。

    4.1 函数定义

    与Python 一样，Scala 也使用 def 关键词来定义一个函数，定义函数一般需要确定函数的名称、参数和函数体，具体格式如下:

    def <fun_name>(<param>: <type>[,...]): type = <expression>

    如果有参数，需要说明参数类型，如果函数不是递归的，就不一定需要指明返回类型。Scala编译时可以根据等号右侧的表达式的类型推导出返回类型。

    如果函数体有多条语句，可用放在 {} 里，最后一句的值为函数返回值，有时需要用 return 显式指明返回值，并退出函数。

    scala> def fun1(x:Int) = if(x > 0) x else -x
fun1: (x: Int)Int

scala> fun1(-100)
res0: Int = 100

// 指明函数返回类型
scala> def fun2(x:Int) :Int= if(x > 0) x else -x
fun2: (x: Int)Int

scala> fun2(-10)
res1: Int = 10

    当然参数不是必须的，有时候函数没有参数。

    scala> def f1() = "hello"
f1: ()String

scala> def f2 = "ok"
f2: String

scala> def f3:String = "scala"
f3: String

    4.2 匿名函数

    Python 中 lambda 函数，Scala 有对应的匿名函数，匿名函数是 Scala 的重要特点之一，其语法格式为:

    ([<param1>:<type>,...]) => <expression>
    scala> (x:Int) => x + 2
res2: Int => Int = <function1>

scala> res2(10)
res3: Int = 12

// 将函数赋值给变量 a
scala> val a = (x:Int) => x + 2
a: Int => Int = <function1>

scala> a(10)
res4: Int = 12

scala> val lst1 = List(2,5,7,8)
lst1: List[Int] = List(2, 5, 7, 8)

scala> lst1.filter((x:Int) => x > 6)
res6: List[Int] = List(7, 8)

// 如果一个参数，只使用一次，可用划线(_) 代替参数，并只要表达式
scala> lst1.filter(_ > 6)
res7: List[Int] = List(7, 8)

    4.3 递归函数

    函数也是对象，可以把参数视为表达式块，而表达式块是可以递归或嵌套，所以函数本身也可以递归或嵌套，以下也可以递归或嵌套。

    scala> // 计算n!
def fn(n: Int): Int={
    if(n <= 0) 1
    else n*fn(n-1)
}
fn: (n: Int)Int

scala> fn(3)
res8: Int = 6

scala> fn(5)
res9: Int = 120

    但是递归会造成堆栈的大量占用，可以使用尾递归进行优化。

    优化的方式是，在函数定义前加上 @annotation.tailrec 这个函数注解(注不是注释) 来声明尾递归，当编译器检测一个函数调用时尾递归时，它就覆盖当前的活动记录而不是在栈中去创建一个新的。只有最后一个语句为递归调用的函数才能由 scala 编译器完成尾递归优化，否则编译时将报错，还是以上例来说明，这是函数 fn 需要做些调整。

    scala>
@annotation.tailrec
def fn(n:Int, m:Int):Int={
    if(n <= 0) m
    else fn(n - 1, m * n)
}

fn: (n: Int, m: Int)Int

scala> fn(5, 1)
res10: Int = 120

    4.4 有默认值参数的函数

    Python 有带默认值参数的函数，Scala 中也有类似函数，使用了默认参数，你在调用函数的过程中可以输入该参数，这时函数就会调用它的默认参数值，如果传递了参数，则传递值会取代默认值。举例如下:

    scala>
def addInt(x:Int, y:Int=0): Int = {
    x + y
}
addInt: (x: Int, y: Int)Int

scala> addInt(20,30)
res0: Int = 50

    4.5 变长参数的函数

    在介绍 Python 函数时，我们介绍了变长参数的函数，函数定义格式为:

    def 函数名(参数1, 参数2, ..., *tupl, **dict)

    在Scala中也有类似功能的函数，而且更简洁，只需要在该函数参数类型后增加一个星号(*)即可，举例如下:

    scala>
def sum(items:Int*):Int={
    var sum = 0
    for(i <- items) sum += i
    sum
}
sum: (items: Int*)Int

scala> sum(1,2,3)
res1: Int = 6

scala> sum(1,2,3,4,5)
res2: Int = 15

    Python 可变参数可输入列表或字典等，以上方式将报错。此时，我们可以在调用函数参数后面追加: _*，_* 操作符将告诉编译器把序列(Aarry，List，Seq，Vector 等）中的每个元素作为一个单独的参数传给函数，具体使用举例如下:

    scala> val lst1 = List(1, 2, 3, 4, 5)
lst1: List[Int] = List(1, 2, 3, 4, 5)

scala> sum(lst1: _*)
res3: Int = 15

    4.6 部分应用的函数

    调用函数时，通常需要指定函数中的所有参数(含默认值的参数除外)，如果参数较多，而且调用时希望保留部分参数值，而修改部分参数值，该如何操作呢？

    这时我们可以在原函数基础上定义一个部分应用函数即可，具体实现请看下例。

    scala> def additems(x:Int, y:Int): Int = x + y
additems: (x: Int, y: Int)Int

scala> val additems10 = additems(10, _:Int)
additems10: Int => Int = <function1>

scala> additems10(30)
res4: Int = 40

    4.7 柯里化函数

    实现部分应用函数，还有一种更简便的方法，采用多个参数表的函数，此时，不是将一个参数表分成应用参数和非应用参数，而是应用其中一个参数表的中参数，不用另一个参数表中参数，这种技术称为函数的柯里化(currying)，具体实现如下。

    scala> def additems(x:Int)(y:Int): Int = x + y
additems: (x: Int)(y: Int)Int

scala> val additems10 = additems(10)_
additems10: Int => Int = <function1>

scala> additems10(30)
res5: Int = 40

    4.8 遍历集合成员

    在数据处理中经常一些函数遍历集合(如 Array、List、Tuple、Set、Map 等）中每个元素，函数遍历每个元素得到一个新的集合或没有返回值。经常使用的函数有 map、filter 、foreach、reduce 等。这些函数在 Python 也有，称为内置函数。

    scala> val list2 = List.range(0, 10, 2)
list2: List[Int] = List(0, 2, 4, 6, 8)

scala> list2.map(i => i + 100)
res6: List[Int] = List(100, 102, 104, 106, 108)

scala> list2.foreach(i => i + 100)  // 与map类似，但没有返回值

scala> list2.filter(i => i > 2)
res8: List[Int] = List(4, 6, 8)

scala> list2.reduce((x, y) => x + y)
res9: Int = 20

    上例中集合为一个列表，如果是 map，又会有哪些函数呢？

    scala> val m1 = Map(1 -> "scala", 2 -> "python", 3 -> "spark")
m1: scala.collection.immutable.Map[Int,String] = Map(1 -> scala, 2 -> python, 3 -> spark)

scala> val m2=m1.filterKeys(_ > 1)
m2: scala.collection.immutable.Map[Int,String] = Map(2 -> python, 3 -> spark)

scala> m1.foreach(x => println(s"key is: ${x._1}, value is: ${x._2}"))
key is: 1, value is: scala
key is: 2, value is: python
key is: 3, value is: spark

    5. 类和对象

    我们在介绍 Python 时，也谈到对象和类，实际上这些都是对代码的封装，函数也是对代码的封装，只是类和对象，封装的层次更高一点，除了变量还包括函数等，是否还有比类或对象更高的封装？包应该就是之一。

    类是对象的抽象，而对象是类的具体实例。类是抽象的，不占用内存，而对象是具体的，占用存储空间。类是用于创建对象的蓝图，它是一个定义包括在特定类型的对象中的方法和变量的软件模板。

    5.1 类和对象简介

    我们可以用 class 定义类，用 new 关键字来创建类的对象，实例如下:

    class Point(xc:Int, yc:Int){
    var x:Int = xc
    var y:Int = yc

    def move(dx:Int, dy:Int){
        x = x + dx
        y = y + dy
        println("x 的坐标点：" + x)
        println("y 的坐标点：" + y)
    }
}

    Scala 中的类不声明为 public，一个 Scala 源文件中可以有多个类。

    以上实例的类定义了两个变量 x 和 y ，一个方法: move，方法没有返回值。

    Scala 的类定义可以有参数，称为类参数，如上面的 xc 和 yc，类参数在整个类中都可以访问。

    接着我们可以使用 new 来实例化类，并访问类中的方法和变量:

    import java.io._

class Point(xc:Int, yc:Int){
    var x:Int = xc
    var y:Int = yc

    def move(dx:Int, dy:Int){
        x = x + dx
        y = y + dy
        println("x 的坐标点：" + x)
        println("y 的坐标点：" + y)
    }
}

object Test{
    def main(args: Array[String]){
        val pt = new Point(10,20)
        pt.move(10, 10)    
    }
}

    5.2 类的继承

    Scala 继承一个基类跟 Java 很相似, 但我们需要注意一下几点:

    • 重写一个非抽象方法必须使用 override 修饰符

    • 只有主构造函数才可以往基类的构造函数里写参数

    • 在子类中重写超类的抽象方法时，你不需要使用 override 关键字

    新建一个文件 test0.scala，输入以下内容:

    import java.io._

class Point(val xc:Int, val yc:Int){
    var x:Int = xc
    var y:Int = yc

    def move(dx:Int, dy:Int){
        x = x + dx
        y = y + dy
        println("x 的坐标点：" + x)
        println("y 的坐标点：" + y)
    }
}

class Location(override val xc:Int, override val yc:Int,
    val zc:Int) extends Point(xc, yc){

    var z:Int = zc
    def move(dx:Int, dy:Int, dz:Int){

        x += dx
        y += dy
        z += dz

        println("x 的坐标点：" + x)
        println("y 的坐标点：" + y)
        println("z 的坐标点：" + z)
    }
}

object Test{
    def main(args: Array[String]){

        val loc = new Location(10,20,15)
        loc.move(10,10,5)
    }
}

    执行程序:

    hadoop@master:~/Projects/ScalaDemos$ scalac test0.scala
hadoop@master:~/Projects/ScalaDemos$ scala Test
x 的坐标点：20
y 的坐标点：30
z 的坐标点：20

    Scala 使用 extends 关键字来继承一个类。实例中 Location 类继承了 Point 类。Point 称为父类(基类)，Location 称为子类。

    override val xc 为重写了父类的字段。

    继承会继承父类的所有属性和方法，Scala 只允许继承一个父类。

    scala 重写一个非抽象方法，必须使用 override 修饰符。

    class Person {
  var name = ""
  override def toString = getClass.getName + "[name=" + name + "]"
}

class Employee extends Person {
  var salary = 0.0
  override def toString = super.toString + "[salary=" + salary + "]"
}

object Test extends App {
  val fred = new Employee
  fred.name = "Fred"
  fred.salary = 50000
  println(fred)
}

// 输出 Employee[name=Fred][salary=50000.0]

    5.3 单例对象

    在 Scala 中，是没有 static 这个东西的，但是它也为我们提供了单例模式的实现方法，那就是使用关键字 object。

    Scala 中使用单例模式时，除了定义的类之外，还要定义一个同名的 object 对象，它和类的区别是，object 对象不能带参数。

    当单例对象与某个类共享同一个名称时，他被称作是这个类的伴生对象: companion object。你必须在同一个源文件里定义类和它的伴生对象。类被称为是这个单例对象的伴生类: companion class。类和它的伴生对象可以互相访问其私有成员。

    import java.io._

    var y:Int = yc

    def move(dx:Int, dy:Int){
        x = x + dx
        y = y + dy

        println("y 的坐标点：" + y)
    }
}

object Test{
    def main(args: Array[String]){
        val pt = new Point(10,20)
        print(Point)

        def printPoint{
            println("x 的坐标点:" + pt.x)
            println("y 的坐标点：" + pt.y)
        }
    }
}

    5.4 伴生对象实例

    // 私有构造方法
class Marker private(val color:String){
    println("创建" + this)    
    override def toString(): String = "颜色标记: " + color
}

// 伴生对象，与类共享名字，可以访问类
object Marker{
    private val markers:Map[String, Marker] = Map(
        "red" -> new Marker("red"),
        "blue" -> new Marker("blue"),
        "green" -> new Marker("green")
    )
    def apply(color:String)={
        if(markers.contains(color)) markers(color) else null
    }
    def getMarker(color:String)={
        if(markers.contains(color)) markers(color) else null
    }
    def main(args:Array[String]){
        println(Marker("red"))
        println(Marker getMarker "blue")  // 单例函数调用，省略了.(点)符号
    }
}
