    2.1 ndarray 对象

    2.1.1 array

    • 示例1
    import numpy as np

a = np.array([1, 2, 3, 4])
b = np.array([1, 2, 3, 4])

c = np.array([[1, 2, 3], [2, 3, 4]])

    • 示例2
    import numpy as np

a = np.array([1,2,3,4])
b = np.array([1,2,3,4])
c = np.array([[1,2,3], [2,3,4]])
c.shape = -1,2

print(c)
print("shape of c : ", c.shape)

    输出:

    [[1 2] 
 [3 2] 
 [3 4]]

shape of c :  (3, 2)

    注意: 当 c.shape 中有一个 -1 值，就会自动补齐。以 -1,2 中的 2 为准，表示2列。

    我们还可以使用reshape来改变数组维度。

    import numpy as np

c = np.array([[1,2,3], [2,3,4]])

n = c.reshape(2,3)
print(n)

n[0][0] = 10
print(n)
print(c)

    可以看到 n 和 c 共享一个内存，当改变 n 中的数组元素的值时，c 也也会相应地改变。

    2.1.5 定义结构体数组

    import numpy as np

person = np.dtype({"names": ["name","age"], "formats": ["S32", "i"]})
print(person)  # [('name', 'S32'), ('age', '<i4')]

a = np.array([("zhang",12)], dtype=person)
print(a)  # array([(‘zhang’, 12)], dtype=[(‘name’, ‘S32’), (‘age’, ‘<i4')])

    结构数组除了通过 a[0][0] 访问外，还可以通过 key 来访问数组。

    print(a[0]['name'])  # zhang

    2.2.1 标准库和 ufunc库计算数据的使用时间

    • 标准库和 ufunc库计算数据的使用时间
    from time import time
import math

x = [i * 0.001 for i in xrange(1000000)]
start = time()

for i, t in enumerate(x):
    x[i] = math.sin(t)

print(time() - start)  # 0.388828039169
    • numpy 库的计算时间
    import numpy as np
from time import time

x = [i * 0.001 for i in xrange(1000000)]
x = np.array(x)

start = time()
t = np.sin(x)
print(time() - start)  # 0.0440118312836

    可见，使用 numpy 库进行数据运算，其效率远远大于标准库。

    4.1 非线性方程组

    4.1.1 方程一

    $$
    \begin{cases}
    5x_1 -25 = 0
    \\ 5x_0^2 - x_1x_2 = 0
    \\ x_0x_1 - 27 = 0
    \end{cases}
    $$

    from scipy.optimize import fsolve

def func(x):
    x0, x1, x2 = x.tolist()
    return [
        5 * x1 - 25,
        5 * x0 * x0 - x1 * x2,
        x0 * x2 -27
    ]

r = fsolve(func, [1, 1, 1])
print(r)  # [3.  5.  9.]

    4.1.2 方程二

    $$
    \begin{cases}
    5x_1 + 3 = 0
    \\ 5x_0^2 - 2\sin x_1x_2 = 0
    \\ x_1x_2 - 1.5 = 0
    \end{cases}
    $$

    import math
from scipy.optimize import fsolve

def func(x):
    x0, x1, x2 = x.tolist()
    return [
        5 * x1 + 3,
        5 * x0 * x0 - 2 * math.sin(x1 * x2),
        x0 * x2 -1.5
    ]

r = fsolve(func, [1, 1, 1])
print(r)  # [0.27061834    -0.6    5.5428616]

    4.1.3 方程三

    由于 fsolve 函数在调用函数 f 时，传递的参数为数组，因此如果直接使用数组中的元素计算的话，计算速度将会有所降低，因此这里先用 float 函数将数组中的元素转换为 Python 中的标准浮点数，然后调用标准 math 库中的函数进行运算。

    在对方程组进行求解时，fsolve会自动计算方程组的雅可比矩阵，如果方程组中的未知数很多，而与每个方程有关的未知数较少时，即雅可比矩阵比较稀疏时，传递一个计算雅可比矩阵的函数将能大幅度提高运算速度。笔者在一个模拟计算的程序中需要大量求解近有 50 个未知数的非线性方程组的解。每个方程平均与 6 个未知数相关，通过传递雅可比矩阵的计算函数使计算速度提高了 4 倍。

    import math
from scipy.optimize import fsolve

def func(x):
    x0, x1, x2 = x.tolist()
    return [
        5 * x1 + 3,
        5 * x0 * x0 - 2 * math.sin(x1 * x2),
        x0 * x2 -1.5
    ]

def j(x):

    x0, x1, x2 = x.tolist()
    return [
        [0, 5, 0],
        [10 * x0, -2 * x2 * math.cos(x1 * x2), -2 * x1 * math.cos(x1 * x2)],
        [0, x2, x1]
    ]

r = fsolve(func, [1,1,1], fprime=j)
print(r)  # [0.27061834    -0.6    5.5428616]

    代码解析: 雅可比矩阵其实就是方程求偏导

    4.2.3 二重积分

    • dblquad : 二重积分

    • trlquad : 三重积分

    球的体积:
    $$
    x^2 + y^2 + z^2 = 1

# -1, 1: 表示 x 区间

    4.3 常微分方程组

    4.3.1 洛伦兹轨迹简介

    scipy.integrate 库提供了数值积分和常微分方程组求解算法 odeint。下面让我们来看看如何用 odeint 计算洛仑兹吸引子的轨迹。

    洛仑兹吸引子由下面的三个微分方程定义:
    $$
    \frac {dx}{dy} = \alpha (x - y)
    \\ \frac {dy}{dt} = x (\rho - z) - y
    \\ \frac {dz}{dt} = xy - \beta z
    $$

    在下面程序中 $\alpha=p$，$\rho=r$，$beta=b$

    洛仑兹吸引子的详细介绍: http://bzhang.lamost.org/website/archives/lorenz_attactor

    这三个方程定义了三维空间中各个坐标点上的速度矢量。从某个坐标开始沿着速度矢量进行积分，就可以计算出无质量点在此空间中的运动轨迹。其中 $\alpha$、$\rho$、$\beta$ 为三个常数，不同的参数可以计算出不同的运动轨迹: x(t), y(t), z(t)。当参数为某些值时，轨迹出现混沌现象: 即微小的初值差别也会显著地影响运动轨迹。

    4.3.2 代码实现

    from scipy.integrate import odeint
import numpy as np

# 定义一个lorenz函数，它的任务是计算出某个位置的各个方向的微分值，
# 这个计算直接根据洛仑兹吸引子的公式得出
def lorenz(w, t, p, r, b):
    x, y, z = w
    return np.array([p * (y - x), x * (r - z), x * y - b * z])

t = np.arange(0, 30, 0.01)
track1 = odeint(lorenz, (0.0, 1.00, 0.0), t, args=(10.0, 28.0, 3.0))
track2 = odeint(lorenz, (0.0, 1.01, 0.0), t, args=(10.0, 28.0, 3.0))

from mpl_toolkits.mplot3d import Axes3D
import matplotlib.pyplot as plt

fig1 = plt.figure()
fig2 = plt.figure()
ax1 = Axes3D(fig1)
ax2 = Axes3D(fig2)
ax1.plot(track1[:, 0], track1[:, 1], track1[:, 2])
ax2.plot(track2[:, 0], track2[:, 1], track2[:, 2])

ax1.set_title("Fig 1")
ax2.set_title("Fig 2")
plt.show()

    调用 odeint，对微分方程求解，odeint 有许多参数，这里用到的四个参数分别为:

    • lorenz : 计算某个位移上的各个方向的速度(位移的微分)

    • (0.0,1.00,0.0) : 位移初始值。计算常微分方程所需的各个变量的初始值

    • t : 表示时间的数组，odeint 对于此数组中的每个时间点进行求解，得出所有时间点的位置

    • args=(10.0,28.0,3.0) : 表示 (p, r, b)，这些参数直接传递给 lorenz 函数
