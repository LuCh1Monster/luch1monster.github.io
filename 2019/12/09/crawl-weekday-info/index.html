    由于工作中会涉及到当前日前时候为工作日，且这个变量会在很多模型中使用，这里整理一下使用 Requests 通过 API 获取当前日期是否为工作日。

    1. goseek API

    goseek API 提供了未来一年的工作日信息，参考网址: http://api.goseek.cn/ 。

    【例子】可以浏览器通过访问 http://api.goseek.cn/Tools/holiday?date=20170528 ，会得到一个返回值 {"code":10000, "data":1}，其中 data 的属性值代表了当前日期是否是公布工作日，该属性会有4种返回值:

    • 0 表示正常工作日
    • 1 表示法定节假日
    • 2 表示节假日调休补班
    • 3 休息日

    但是使用这个 API 需要注意以下几个问题:

    • Goseek API 是每年更新一次，也就是可能在 2019-12-09 的时候，未来一年的信息已经更新好了，也有可能没有更新。所以，在使用该方法时一定要 Check 未来一年的信息是否已经更新。
    • 只支持 dt 查询，不支持按月份进行查询
    def crawl_day_info_goseek(dt):
    """
    根据传入的日期，返回是否是工作日。
    :param dt: 日期
    :return: 返回值分4种
        0 表示正常工作日
        1 表示法定节假日
        2 节假日调休补班
        3 休息日
    """
    api_uri = "http://api.goseek.cn/Tools/holiday?date=%s" % dt
    req = request.Request(api_uri)
    response = request.urlopen(req)
    day_info = json.loads(response.read())
    return day_info

    2. Easybots

    easybots API 提供了未来一年的工作日信息，参考网址: http://www.easybots.cn/holiday_api.html ，网站上提供了具体使用的几种方式。由于本次更新 2020 年工作日表信息时，goseek 的 API 还没有更新，这里选择 easybots 进行数据的更新。

    优点:

    • 支持 dt 查询
    • 支持按月份查询

    缺点:

    • 该网站上明确说明是收费的，如果没有购买，得到的数据会是一堆垃圾数据。但是其实这里测试下来，不够买也是可以获取数据，且数据也都是正确的。
    • 后续使用时，还是需要 Check 一下，以防后台做了更改，得不到正确的数据。

    【例子】可以浏览器通过访问 http://www.easybots.cn/api/holiday.php?m=202001 ，会得到一个整个月份的信息:

    {"202001":
        {"01":"2", "04":"1", "05":"1",
         "11":"1", "12":"1", "18":"1",
         "24":"1", "25":"2", "26":"2","27":"2",
         "28":"1","29":"1","30":"1"
        }
}

    其中每个月份下会有具体日期，其返回值就是是否是工作日的信息。该 API 返回具体日期信息只有3种值:

    • 0 表示正常工作日
    • 1 表示休息日
    • 2 表示节假日

    这里在程序中进行了处理，工作日为 1，非工作日均为 0。

    def crawl_day_info_easybots(month, dt_start, dt_end):
    """
    根据传入的日期，返回是否是工作日。
    :param month: 月份，比如 202001 表示2020年1月份
    :param dt_start: 月初1号
    :param dt_end: 月末
    :return: 返回一个 DataFrame，包含这个月每一天的信息
    """
    api_uri = "http://www.easybots.cn/api/holiday.php?m=%s" % month
    req = request.Request(api_uri)
    response = request.urlopen(req)
    month_info = json.loads(response.read())

    data = []
    for d in pd.date_range(start=dt_start, end=dt_end, freq='D'):
        dt = d.strftime('%Y-%m-%d')
        num = "0" + str(d.day) if d.day < 10 else str(d.day)
        # 工作日为 1，非工作日为 0
        isweekday = 1 if int(month_info.get(month).get(num, 0)) == 0 else 0
        data.append((dt, isweekday))

    df_month = pd.DataFrame(data, columns=['dt', 'is_weekday'])

    return df_month

    3. 获取2020整年的信息

    # 获取2020数据
df_list = []
for m in pd.date_range(start='2020-01-01', end='2021-01-01', freq='M'):
    api_month_fmt = m.strftime('%Y%m')  # 传入API获取数据的月份格式
    month_start = m.strftime('%Y-%m') + '-01'
    next_month_start = m.strftime('%Y-%m-%d')
    df_info = crawl_day_info_easybots(month=api_month_fmt, dt_start=month_start, dt_end=next_month_start)
    df_list.append(df_info)

    df = pd.concat(df_list, ignore_index=True)
    df.to_csv('weekday_info.csv', index=False, encoding='utf8')
    df.to_excel('weekday_info.xlsx', index=False, encoding='utf8')

    def update_impala_table(df):
    """
    更新Impala工作日表
    :return:
    """
    try:
        conn = connect(host='', port='', auth_mechanism='PLAIN',
                user='', password='')
    except Exception:
        raise Exception('Impala连接存在问题')
    cur = conn.cursor()
    values_list = []

    for raw in df.values:
        value = '{!r}'.format(list(raw))
        value = '(' + value.strip('[').strip(']') + ')'
        values_list.append(value)

    values = ','.join(x for x in values_list)

    insert_sql = r"""insert into db.weekday values %s""" % values
    cur.execute(insert_sql)
    print("数据更新完成")
