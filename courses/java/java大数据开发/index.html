    1. JDK概述、开发环境介绍及基本语法

    1.1 Java语言简介、特点、工作原理

    1.1.1 Java语言简介

    Java是 Sun(Standford University Network) 公司推出的一种编程语言。

    它是一种通过解释方式来执行的语言，语法规则和 C++ 类似。

    同时，Java 也是一种跨平台的程序设计语言。1995/5/23，Java Development Kits(当时的JDK全名) 1.0a2版本正式对外发表。

    Java 最初的设计意图是在消费类电子产品上做嵌入式开发 www 发展起来之后，开始向 web方向发展，并取得巨大成功。

    目前在移动开发领域，Java 也是最重要的开发语言之一。

    1.1.2 工作原理

    Java 编写的程序可以在不同平台上运行，这也是 Java 语言的一大优势。

    • 通过“编译器”，将 java 源程序编译成 java 字节码文件(.class)

    • 通过“解释器”，将读取 java 字节码文件，并运行

    1.1.3 Java语言特点

    • 安全性

    • 多线程

    • 可移植性

    • 分布性

    • 高性能(相对于其他解释性语言)

    • 健壮性

    1.2 Java平台组成、安装方法

    1.2.1 Java平台组成

    Java ME(Micro Edition 微缩版，Java ME 3) 嵌入式开发: 嵌入式小程序、只能手机游戏

    Java SE(Standard Edition 标准版，Java SE 6) 桌面程序开发

    • ATM自动取款机程序

    • 桌面程序、Java EE 客户端程序

    Java EE(Enterprise Edition 企业版，Java EE 5) 主要用于分布式的网络程序的开发

    • 移动、电信等通信行业的大型系统

    • 银行、税务等金融行业的大型系统

    • 政府的电子政务系统

    • 大型企业用 ERP 系统

    1.3.1 Java程序结构

    • package: 零个或多个，必须放在文件开始
    • import: 零个或多个，必须放在所有类定义之前
    • public ClassDefinition: 零个或多个

    • interfaceDefinition: 零个或多个
    • 类个数: 至少一个类，最多只能有一个 public 类
    • 源文件命名: 若有 public 类，源文件必须按该类命名
    • 标识符: 区分大小写

    1.3.3 Java数据类型

    1) 基本数据类型

    所有基本类型所占的位数都是确定的，并不因为操作系统的不同而不同。

    所有基本类型的关键词都是小写的。

    数据类型 所占位数 数的范围
    char 16 0 ~ 65535
    byte 8 $-2^7$ ~ $2^7-1$
    short 16 $-2^{15}$ ~ $2^{15}-1$
    int 32 $-2^{31}$ ~ $2^{31}-1$
    long 64 $-2^{63}$ ~ $2^{63}-1$
    float 32 $2^{-149}$ ~ $2^{128}-1$
    double 64 $2^{-1074}$ ~ $2^{1024}-1$

    2) 布尔型

    • 布尔类型数据只有两个值 true 和 false，且他们不对应于任何整数值。布尔型变量的定义如: boolean b = true;

    • 布尔型数据只能参与逻辑关系运算: && 、|| 、==、!=、!

    示例:

    boolean b1;
boolean b2 = true;
b = !b2;
boolean b = (b1 && b2) != false;

    3) 字符型

    • 字符型数据代表 16 位的 Unicode 字符

    • 字符常量是用单引号括起来的一个字符，如: 'a'、'B'、'\n'、'\u0030'

    • 字符型数据的取值范围为: 0 ~ 65535 或者 \u0000~\uFFFF。其中，\u0000 为缺省值

    示例:

    char c1;        // 缺失值为0
char c2 = '0';  // 赋初值为字符'0'
char c3 = 32;   // 用整数赋初值为空格

    4) 整型

    整数常量:

    • 十进制整数，如 123、-456、0

    • 八进制整数，以 0 开头，如 0123 表示十进制 83，-011 表示十进制数 -9

    • 十六进制整数，以 0x 或 0X 开头，如 0x123 表示十进制 291，-0X12 表示十进制数 -18

    整型变量:

    • 类型为 byte、short、int 或 long

    • byte 在机器中占 8 位，short 占 16 位，int 占 32 位，long 占 64 位

    整型变量的定义示例:

    int x = 123;  // 指定变量x为int型，且赋初值为123
byte b = 8;
short s = 10;
long y = 123L;
long z = 123l;

    5) 浮点型

    十进制数形式，由数字和小数点组成，且必须有小数点，如: 0.123、.123、123.0。

    科学计数法形式，如: 123e3 或 123E3，其中 e 或 E 之前必须有数字，且 e 或 E 后面的指数必须为整数。

    32位浮点数形式: 0.23f、1.23E-4f、.18F

    1.4 面向对象理论及其Java实现

    1.4.1 类的组成部分

    面向对象技术主要围绕以下几个概念:

    • 对象(Object)
    • 抽象数据类型(ADT)
    • 类(class)
    • 类型层次(子类)
    • 继承性(Inherit)
    • 多态性

    变量: 指对象的所知道的状态

    方法: 指对象的功能单元

    消息: 软件对象通过相互间传递消息来相互作用和通信，一个消息有以下三部分组成

    • 接收消息的对象

    • 接收对象要采用的方法

    • 方法需要的参数

    1.4.2 类的定义

    以一个例子来进行说明，新建一个文件 Hello.java，输入内容如下:

    class Hello{
     private String s;
     public void showString(){
         System.out.println(s); 
     }
     public void changeString(String str){
         s = str; 
     }
}

    在程序中所操作的对象是类的一个实例。创建一个对象 Hello obj = new Hello();，调用方法 obj.showString();。

    完善文件 Hello.java 程序:

    public class Hello{
  private String s;

  public void showString(){
    System.out.println(s);
  }

  public void changeString(String str){
    s = str;
  }

  public static void main(String[] args){
    Hello obj = new Hello();
    obj.changeString("Apache Hadoop");
    obj.showString();
  }
}

    运行结果:

    → java Hello
Apache Hadoop

    1.4.3 类的继承

    • 类与类之间还有一种父与子的关系，子继承父类(超类)方法与属性，这个我们就称之为继承，继承体现了类之间的泛化关系
    • 子类拥有父类的一些公有的属性和方法，并且子类可能也会有自己的方法和编程
    • 除了添加父类中没有的新方法外，子类还可以通过重写父类的方法重定义自己的行为方式

    1) 继承的语法

    [类的修饰符] class <子类名> extends <父类名>{
     <属性定义>;
     <方法定义>;
}
    • 在定义父类时用 extends 关键字指明新定义的父类，就在两个类之间建立了继承关系。新定义的类称为子类，它可以从父类那里自动继承所有非 private 的属性和方法作为自己的成员
    • 子类永远不会继承父类的构造方法。因为构造方法永远和类名相同
    • 继承了什么？除了构造方法之外，父类的所有非 private 方法和非跨包的默认属性都由子类的对象继承

    1.4.4 字符串

    构建 String 实例对象的方法主要有4种:

    • 采用双引号将 Java 字符序列括起来，即通过构造字符串直接赋值的方式构造 String 实例对象。例如: String s1 = "abcd";

    • 采用 new 运算符。例如: String s1 = new String();

    • 通过运算符 + 或其他成员方法生成 String 实例对象。例如: String s1 = "abcd" + "1234";

    • 通过给构造方法传递一个字符数组创建一个非空串

    char chars[] = {'a', 'b', 'c'};
String s = new String(chars);

    1) 字符串String的主要方法

    • length: 此方法返回字符串的字符串个数

    • charAt: 该方法返回字符串中 index 位置上的字符，其中 index 值的范围是 0~length-1

    • getChars: 如果要从字符串中提取一个以上的字符，则可以用此方法

    • concat: 将当前字符串对象与给定的字符串 str 连接起来。返回连接后的字符串

    • indexOf 和 lastIndexOf: 返回字符 ch 在字符串中出现的第一个和最后一个位置。

    为了在给定的字符串中检索特定的字符串或子串，类 String 提供了上面两种方法，并通过方法重写更方便的处理这类问题。

    这两种方法中，如果查找成功，则返回匹配成功的字符位置，如果没有查找到，则都返回 -1。

    public class StringDemo{

        System.out.println(str.replace('p', 'd'));

        char [] arrChar = str.toCharArray();

    }
