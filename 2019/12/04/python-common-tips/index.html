    1. 产生随机密码

    后台修改用户密码时，可以随机地生成一个 n 位的密码，使用 string 和 random 模块可以非常轻松的做到。

    import string
import random

password = ''.join(random.sample(string.ascii_letters + string.digits, 12))
print(password)  # Yd6RJ1g3nGkB

    2. 获取当前文件的路径

    很多时候我们需要获取当前文件的相对路径和绝对路径，这会在我们在项目工程化时大大减少我们的代码量，同时我们也会涉及到一些场景必须使用绝对路径，比如在邮件中添加附件，Python 就必须知道你所要添加的附件的绝对路径是什么。

    通常我们会涉及到以下几种使用场景:

    2.1 获取当前文件的文件名

    import os
print('当前文件的文件名为: ', os.path.abspath(__file__))

    2.2 获取当前文件的父文件夹名

    import os
print('当前文件的父文件夹名为: ', os.path.dirname(os.path.abspath(__file__)))

    2.3 获取当前文件的同级目录下的文件

    import os
fielname = os.path.join(os.path.dirname(os.path.abspath(__file__)), 'data/test.sql')

    值得一提的是，上面的实现是用了 __file__ 属性，这在 Jupyter Notebook 中是无效的。

    3. NumPy 和 Pandas 关闭科学计数显示

    很多时候我们在使用 NumPy 和 Pandas 的时候，会发现某些列上显示的值为科学计数法方式，但其实很多时候我们就是想看一下值大概是多少，这样就很尴尬。我们可以通过设置参数来关闭NumPy 和 Pandas 科学计数法的显示:

    • NumPy
    import numpy as np
np.set_printoptions(suppress=True, threshold=np.nan)

    其中，suppress=True 表示取消科学记数法，threshold=np.nan 完整输出(没有省略号)。

    • Pandas : 保留三位小数
    pd.set_option('float_format', lambda x: '%.3f' % x)

    4. Pandas设置显示最大行数和列数

    pd.set_option('max_columns', 200)
pd.set_option('max_rows', 200)

    5. 自定义Python包添加到环境中

    有时候需要将自己编写的函数添加到环境中，这样可以随时地在其他文件中进行调用，能让你像使用 from sklearn.model_selection import train_test_split 一样调用你自己的函数，一般有两种方法:

    5.1 定义 .pth 文件

    在 site-packages 文件夹下创建一个 mypackage.pth，里面添加自定义包的绝对路径，这样你就可以直接在文件中进行调用。例如你有一个自定义包叫 ipytools，则你需要添加如下代码:

    /home/user/ipytools

    5.2 使用 sys 包

    使用 sys 模块来将自定义包添加到环境中，推荐使用这一种方法，这样更方便管理自定义包的代码。

    import sys
sys.path.append('/home/user/ipytools')

    6. Windows下上传文件到共享文件夹

    假设我们的共享文件夹(Windows服务器)的 IP 是 1.1.1.1，共享文件路径是 \\1.1.1.1\\share-fold\\test1，那么可以参照以下代码进行修改，将想要的文件上传至共享文件夹内:

    import os
import shutil

upload_dir = os.system(r"net use \\1.1.1.1\\share-fold\\test1")
files = ['test/'+ file for file in os.listdir('test') if file.endswith('.csv')]
for file in files:
    shutil.copy(file, upload_dir)

    其中 test 是我们本地的文件夹，里面有大量的 csv 文件，有兴趣的同学还可以加一个进度条显示进度。

    值得注意的是，os 和 shutil 其实是可以在共享文件夹执行创建(删除)文件夹的操作，这样就可以让你代码鲁棒性更高。

    import os

# 检验共享文件夹中是否已经存在此文件夹，如果不存在则创建文件夹，
# 如果存在则先删除再创建
remote_fold = r"\\1.1.1.1\\share-fold\\test1"
if not os.path.exists(remote_fold):  
    os.mkdir(remote_fold)
else:
    shutil.rmtree(remote_fold)
    os.mkdir(remote_fold)

    在使用 Python 的 os.system 将文件上传到共享文件夹时，有时候会出现错误。原因在于共享文件夹名中包含了 & 为，例如 \1.2.3.4\xue\xi&123。直接在 cmd 终端下使用 net use \1.2.3.4\xue\xi&123 也无法进入文件夹下。原因在于，Windows 下 cmd 任务 & 是一个连接两个命令的连字符，所以需要 & 进行转义。进行转义的方法很简单，在 & 前面加上 ^ 变成 ^& 就可以了。

    7. 压缩文件夹

    使用 shutil 可以压缩文件夹，有时候我们也需要只将文件夹中一些文件进行打包压缩，示例代码如下:

    import os
import shutil

def zip_files(archive_file_name='', format="zip", root_dir='', spec_files=[]):
    """
    archive_file_name: 压缩文件的文件名
    format: 压缩格式
    root_dir: 需要压缩的文件夹
    spec_files: 只压缩 root_dir 文件夹指定文件列表
    """
    if root_dir == '':
        raise '指定需要压缩文件夹'
    elif archive_file_name == '':
        raise '指定压缩文件的文件名，无后缀'
    elif format != 'zip':
        raise '只支持zip压缩格式'
    elif spec_files == [] or isinstance(spec_files, list) == False:
        shutil.make_archive(archive_file_name, format, root_dir)
    elif isinstance(spec_files, list) == True and len(spec_files) > 0:
        if os.path.exists('tmp/'):
            shutil.rmtree('tmp')
            os.mkdir('tmp')
        else:
            os.mkdir('tmp') 
        [shutil.copy(os.path.join(root_dir, file), 'tmp/') for file in spec_files]
        shutil.make_archive(archive_file_name, format, 'tmp')
        shutil.rmtree('tmp')

    8. SQLAlchmey连接MySQL

    Python 有很多库支持操作 MySQL ，比如 PyMysql。如果只是涉及对表的读取、写入，这里建立使用 sqlalchemy 和 Pandas 就可以了。

    8.1 连接MySQL

    import pandas as pd
from sqlalchemy import create_engine

conn = create_engine('mysql+pymysql:user:passwd@host:port/testdb?charset=utf-8')
sql = "select * from test limit"
df = pd.read_sql(sql,con=conn)

    值得注意的是，这里 mysql 的连接并没有断开。

    8.2 关闭MySQL的连接

    关于如何正确关闭 sqlalchemy 的连接，网上大多数博客说的都是下面这一种方法:

    conn = create_engine('mysql+pymysql:{user}:{password}@{host}:{port}/testdb?charset=utf-8')
con = conn.connect()
# 你的处理代码
df = pd.read_sql(query_sql, con)
df.to_sql(tb_name, con, if_exists='append', index=False)
con.close()

    但其实这样，也没有真正关掉连接，应该使用 dispose 来关闭连接。

    conn = create_engine('mysql+pymysql:{user}:{password}@{host}:{port}/testdb?charset=utf-8')
df = pd.read_sql(query_sql, conn)
df.to_sql(tb_name, conn, if_exists='append', index=False)
conn.dispose()

    9. Requests+XML测试服务器状态

    如果你的日常任务中涉及到从 API 获取数据，如果服务器状态正态正常，就会返回一个 200 的状态响应码。如果服务器宕机了，就会返回是 500 的状态码。

    import requests

url = "http://www.bbb.com/test/api/"
headers = {"Content-Type": "application/xml", 
           "Accept": "application/xml;charset=UTF-8", 
           "X-KIE-ContentType": "XSTREAM", 
           "Authorization": "qcode"}
xml = """
<?xml version="1.0"?>
<company>123</company>
"""

resp = requests.post(url=url, data=xml.encode("utf-8"), headers=headers)
print(resp.status_code)

    如果打印出来是 200，就表示服务器是没有问题的。

    10.3 PyPrind 简单教程

    直接上代码:

    import pyprind
import time

bar = pyprind.ProgBar(n, monitor=True)
for i in range(n):
    time.sleep(timesleep) # your computation here
    bar.update()
print(bar)
