    1. 安装 MySQL

    首先先确认你已经安装好 MySQL，默认的安装位置是 /var/lib/mysql，且数据文件夹也是在这个位置，并且已经启动过了。

    2 迁移数据文件库

    • 首先关闭掉你的 MySQL 进程，这步非常重要 。
    → /etc/init.d/mysqld stop
    • 迁移数据文件存放地址，必须使用 mv，因为它可以保持文件的原始属性，我们将其迁移到 /home 目录下
    → mv /var/lib/mysql /home/
    • 启动 mysql
    → mysql -uroot -p
Enter password:

    发现输出密码后，出现报错信息:

    ERROR 2002 (HY000): Can't connect to local MySQL server through socket '/var/lib/mysql/mysql.sock' (2)

    解决办法:

    mkdir -p /var/lib/mysql/
ln -s /home/mysql/mysql.sock /var/lib/mysql/mysql.sock

    再次登录就 OK 了。

    • 查看 mysql 库中的数据存放位置
