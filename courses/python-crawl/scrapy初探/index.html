    1. 爬取网页

    1.1 创建豆瓣爬虫项目

    → scrapy startproject douban

    1.2 编写爬虫文件

    📃 douban/spiders/spider.py

    from scrapy.contrib.spiders import CrawlSpider

class Douban(CrawlSpider):

    name = "douban"
    start_urls = ['https://movie.douban.com/top250']

    def parse(self, response):
        print(response.body)
        print(response.url)

    这里就可以直接运行 scrapy crawl douban。

    注意: 这里 scrapy crawl douban 里douban 是 spider.py 中 name 设定的，name 用于设定爬虫的名字。

    如果爬取内容为空，则可以考虑添加 USER-AGENT:

    USER_AGENT = 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_8_3) AppleWebKit/536.5 (KHTML, like Gecko) Chrome/19.0.1084.54 Safari/536.5'

    1.3 添加运行文件

    我们这样每次需要在命令行中输入命令去执行，我们也可以添加一个 main.py，这样就可以在PyCharm中直接运行了。

    📃 douban/main.py

    from scrapy import cmdline
cmdline.execute("scrapy crawl douban".split())

    2. 文件结构

    • items.py : 定义需要抓取并需要后期处理的数据
    • settings.py : 配置文件，修改 user-agent、设定爬取时间间隔、设置代理、配置各种中间件等
    • pipeline.py : 用于存放执行后期数据处理的功能，从而将爬虫的抓取与处理分开

    📃 douban/items.py

    from scrapy import Item, Field

class DoubanItem(Item):
    # name = scrapy.Field()
    title = Field()
    movieInfo = Field()
    star = Field()
    quote = Field()

    📃 douban/pipelines.py

    class DoubanPipeline(object):
    def process_item(self, item, spider):
        return item

    将数据写入到文件中，需要在 settings.py 中设置。

    📃 douban/settings.py

    BOT_NAME = 'douban'

SPIDER_MODULES = ['douban.spiders']
NEWSPIDER_MODULE = 'douban.spiders'

    📃 douban.py

    import scrapy
from scrapy.contrib.spiders import CrawlSpider
from scrapy.http import Request
from scrapy.selector import Selector
from doubanmovie.items import DoubanmovieItem

class DoubanSpider(CrawlSpider):
    name = "douban"
    allowed_domains = ["movie.douban.com/top250"]
    start_urls = ['http://movie.douban.com/top250/']

    url = 'http://movie.douban.com/top250'

    def parse(self, response):
        item = DoubanmovieItem()
        selector = Selector(response)
        Movies = selector.xpath('//div[@class="info"]')

        for eachMovie in Movies:
            title = eachMovie.xpath('div[@class="hd"]/a/span/text()').extract()
            fullTitle = ''
            for each in title:
                fullTitle += each

            movieInfo = eachMovie.xpath('div[@class="bd"]/p/text()').extract()
            star = eachMovie.xpath('div[@class="bd"]/div[@class="star"]/span/em/text()').extract()[0]
            quote = eachMovie.xpath('div[@class="bd"]/p[@class="quote"]/span/text()').extract()

            # quote maybe null, so we should judge first
            if quote:
                quote = quote[0]
            else:
                qoute = ''

            item['title'] = fullTitle
            item['movieInfo'] = ';'.join(movieInfo)
            item['star'] = star
            item['quote'] = quote

            yield item

        # 提取下一页的网址，用于下一页的数据爬取
        nextLink = selector.xpath('//span[@class="next"]/link/@href').extract()

        # 因为这里一共就只有10页，所以我们下一页可能提取到最后是空的，所以作下判断
        # 如果不为空，则将下一页进行爬取
        if nextLink:
            nextLink = nextLink[0]
            print(nextLink)
            yield Request(self.url+nextLink, callback=self.parse)
