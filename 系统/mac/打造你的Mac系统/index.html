    由于近期自己的 Macbook Pro 2017 显示屏出问题了，今天刚拿到维修好的机器。因为之前为了尝鲜 Catalina 系统，结果发现这个系统和 Mojave 一般问题多的很。所以拿到手的第一件事情就是把系统更新回出厂版本 High Sierra 系统，这个版本深得朕心。

    作为一个程序员，系统安装好了之后当然需要装一系列的软件，以及开发环境。这里做一个记录，方便后续再次重新安装系统时，做一个参照。

    1. 重新安装Mac OS系统

    Mac 重新安装 OS 系统有 3 种方式:

    • Command+R : 此模式下可以重新新安装你在 Mac 上安装过的最新 Mac OS 系统版本，但不会升级到更高的版本
    • Option+Command+R : 此模式下可以安装与你的 Mac 兼容的最新 Mac OS 系统版本
    • Shift+Option+Command+R : 此模式下可以安装与你刚购买时的 Mac OS 系统版本

    剩下的操作其实比较一致，对 Macintosh HD 盘进行抹盘，然后点击重新安装系统即可。

    这里我选择了最后一种，因为我的 Mac 购买时就是 High Sierra 版本，且使用过 Mojave 和 Catalina 系统之后，我觉得 High Sierra 最为稳定和流畅。

    6. 安装开发环境

    6.1 安装Homebrew

    Homebrew 的官网 https://brew.sh/ ，百度连个官网都搜索不到，真的是无力吐槽。

    执行以下命令:

    → /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

    如果你有可用 VPN 的话，其实执行上述代码是可以正确安装的。不幸的是，家里用的是垃圾长城宽带，安装过程中会发现 homebrew 安装会出现很多问题，网上有一些实际处理的方案，但并未真正测试过。以下是我安装 homebrew 中出现的一些问题以及解决办法。

    6.1.1 亲测有用的解决方案

    这个解决方案参考了这篇博客 https://segmentfault.com/a/1190000021360086?utm_source=tag-newest ，以下是具体步骤。

    • 先获取 install文件，比如命名为brew_install:
    → curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install >> brew_install
    • 打开 brew_install 文件进行修改:
    # 修改前
BREW_REPO = "https://github.com/Homebrew/brew".freeze

# 修改后
BREW_REPO = "https://mirrors.ustc.edu.cn/brew.git".freeze
    • 保存以后 ruby 执行 brew_install
    → ruby brew_install

    此时脚本会停在:

    ==> Tapping homebrew/core Cloning into ‘/usr/local/Homebrew/Library/Taps/homebrew/homebrew-core’…
    • 把 homebrew-core 更换为中国科大镜像
    → git clone git://mirrors.ustc.edu.cn/homebrew-core.git/ /usr/local/Homebrew/Library/Taps/homebrew/homebrew-core --depth=1

→ cd $(brew --repo)
→ git remote set-url origin https://mirrors.ustc.edu.cn/brew.git

→ cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core"
→ git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git
    • 安装成功、更新、检测
    → brew update
→ brew doctor

    我们顺便使用 homebrew 安装一下 wget:

    → brew install wget

    如果想把中科大的镜像源修改回原来的镜像，可以参考这篇博客 https://www.jianshu.com/p/397860ff2a0e 。

    6.1.2其它尝试过的方案

    • homebrew 安装中总是会出现下载中断的错误，将网络切换到手机热点即可，亲测是有小的。如果你的流量够多的，可以尝试下这个方法。

    • 如果出现 Error: Checksum mismatch. 错误，使用命令行删掉 Already downloaded: 后面文件，即 portable-ruby-2.6.3.mavericks.bottle.tar.gz。建议使用 rm -f 命令，而不要使用 rm -rf 命令。

    # 错误信息

    6.2 安装Git

    Mac 本身就安装了 Git，所以不需要进行安装。

    6.2.1 优化访问GitHub速度

    参考了 https://baijiahao.baidu.com/s?id=1608100091125662190&wfr=spider&for=pc 博客，但是按照该博客的方法，并未找到可以用的 IP，反而博客中提到的 192.30.255.113 这个 IP 配置好之后，访问速度确实快了一些。

    打开 /etc/hosts 文件，在末尾添加以下内容:

    📃 /etc/hosts

    192.30.255.113    github.com

    6.2.2 配置访问无密码访问GitHub

    • 配置 GitHub 用户和邮箱:
    → git config --global user.name "luch1monster"
→ git config --global user.email "442523981@qq.com"
    • 生成 RSA 密钥
    → ssh-keygen -t rsa -C 442523981@qq.com

    将文件 ~/.ssh/id_rsa.pub 内容添加到你的 GitHub 仓库设置里面。

    • 测试是否成功
    → ssh -T git@github.com
Hi LuCh1Monster! You've successfully authenticated, but GitHub does not provide shell access.

    6.3 安装 Node.js

    安装 Node.js 的目的其实是为了使用 npm 命令工具，最简单的方法是使用 homebrew 进行安装:

    → brew install node

    安装完之后可以使用以下命令查看是否安装成功。

    → node -v
v12.16.1
→ npm -v
6.13.4

    还有一个方法就是直接通过下载 Node.js 的安装包来进行安装。Node.js 安装包的下载地址: https://nodejs.org/en/#download ，分为 LTS 和 Current 版本。这里我们选择 LTS 版本进行下载安装。

    这里我们还需要切换一下 npm 的镜像源切换到淘宝镜像源，避免安装时速度太慢。

    → npm config set registry https://registry.npm.taobao.org

    6.4 安装 Anaconda3

    我们通过清华大学的镜像网站来下载 Anaconda，这样会快非常多，地址为 https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/ 。这里我们下载 3.5.2 版本，对应到 Python 的版本是 3.6.5，该版本没有像 3.7 和 3.8 那么多的 bug。

    下载好文件后，执行以下命令:

    → bash Anaconda3-5.2.0-MacOSX-x86_64.sh -p ~/bigdata/anaconda3

    这里加上 -p 是指定 Anaconda 的安装目录，不然它就会安装到根目录下。后续我会在 bigdata 目录下面安装多个语言的开发环境，这样方便后续的环境管理。

    注意，安装过程中，以下是两个选项是你必须要确认的:

    Do you wish the installer to prepend the Anaconda3 install location
to PATH in your /Users/yumingmin/.bash_profile ? [yes|no]
[yes] >>> yes

Do you wish to proceed with the installation of Microsoft VSCode? [yes|no]
>>> yes

    安装完毕后，执行以下命令查看 Python 版本:

    → source ~/.bash_profile
→ python -V
Python 3.6.5 :: Anaconda, Inc.

    6.6 安装 Scala

    Scala官方下载地址: https://www.scala-lang.org/download/

    6.6.1 Homebrew方式安装

    Mac 下安装 scala 最简单的方式就是使用 homebrew 进行安装:

    → brew update
→ brew install scala

    6.6.2 通过环境安装

    下载对应系统的 scala 版本，这里下载的是 scala-2.12.7.tgz ，下载完成后解压。

    → tar zxvf scala-2.12.7.tgz -C ~/bigdata/
→ mv scala-2.12.7 scala2.12.7

    6.6.3 添加环境变量

    在 ~/.bash_profile 添加 scala 的系统环境变量:

    # scala settings
export SCALA_HOME="/Users/yumingmin/bigdata/scala2.12.7"
export PATH=${PATH}:${SCALA_HOME}/bin

    执行 source ~/.bash_profile ，scala 就安装完成了。
