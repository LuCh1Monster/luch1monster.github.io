    将DataFrame添加到邮件正文

    郁明敏 Python/ Python科学计算/ Pandas
    创建时间:2019-12-12 19:30
    字数:219 阅读:

    有时候我们会需要将做好的表(Datafrome) 添加到邮件正文中，以此来做一个邮件监控。

    from jinja2 import Template
import pandas as pd
import numpy as np

df = pd.DataFrame(np.random.randn(10, 4), columns=list('ABCD'))

html = "<p>我们要将表格数据放到邮件正文中。</p>"

html_template = '''
            <table border="1">
            <thead>
                <tr>
                {\% for header in df.columns \%}
                    <th  bgcolor="#7fffd4">{{ header }}</th>
                {\% endfor \%}
                </tr>
            </thead>
            <tbody>
                {\% for row in df.values \%}
                    <tr>
                    {\% for v in row \%}
                        <td>{{ v }}</td>
                    {\% endfor \%}
                    </tr>
                {\% endfor \%}
            </tbody>
            </table>
        '''
template = Template(html_template.replace('\n', ''))
html += template.render(df=df)
send_mail(success_mail_list, '发送带有表格数据的邮件', html)
